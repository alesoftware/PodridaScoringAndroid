# Podrida Scoring Android

An Android application for scoring Podrida card games, built with Kotlin and Python (Chaquopy).

## Project Structure

This is an Android application that uses:
- **Kotlin** for the Android native code
- **Python** (via Chaquopy) for backend logic
- **Flask** for web-based UI components
- **Google Sheets API** for data persistence

## Getting Started

### Prerequisites

- Android Studio
- Android SDK
- Python 3.8+

### Building the Project

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Build and run on your device or emulator

## Features

- Game scoring and tracking
- Tournament management
- Player management
- Google Sheets integration for data storage
